# 📝 Blog Application

A full-stack blog application built using Node.js, Express.js, EJS, and PostgreSQL.

## 🚀 Features
- Create, read, update, and delete blog posts
- Server-side rendering using EJS
- Structured routing with Express.js
- PostgreSQL database integration
- Clean and organized backend architecture

## 🛠 Tech Stack
- Node.js
- Express.js
- EJS
- PostgreSQL
